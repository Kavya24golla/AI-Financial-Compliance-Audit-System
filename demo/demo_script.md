# 🎬 Demo Script

1. Open the application in the browser (frontend dashboard).

2. Select a company from the dropdown or input field.

3. Click on **Run Analysis** (or trigger the pipeline).

4. Show the system loading / processing state.

5. Once completed:
   - Display the **financial metrics / ratios**
   - Show the **risk score**

6. Highlight the **detected anomalies**:
   - Point out key issues (e.g., high debt ratio, low liquidity)

7. Show the **final decision output**:
   - Approve / Review / Escalate

8. Open or highlight the **AI-generated explanation**:
   - Explain how the system justifies the decision using RAG + LLM

9. Navigate to the **audit logs / trace section**:
   - Show that each step (retrieval → analysis → compliance → decision) is logged

10. (Optional) Refresh or run another company to show consistency

11. Conclude:
   - “The system automates financial analysis, compliance validation, and generates explainable decisions with full audit traceability.”
